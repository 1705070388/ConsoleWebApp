Project On Console

Visual Studio is a desktop app. It is used to create websites. If you want the app on your desktop and start working on it, then you have come to the right place.
Step 1: Go to the following link and download the one that matches your system the best
URL: https://code.visualstudio.com/Download/
Step 2: Setup Visual Studio on your system. And open it
Step 3: Go to file manager and create a new folder and open it in Visual Studio. For that, you need to click on the file option in Visual Studio and click on Open folder and open your folder.
Step 4: To create a new file, right click on the empty space on your left side and click on New File. You can name the file.
Step 5: Start writing code in the centre.







Javascript projects can be done on the console screen. To open the console screen, you need to right click on the website. Then, click on Inspect. You will see some tabs. Click on the Console tab. And you are in the console screen!
